Guía rápida

Descripción

Este repositorio está preconfigurado para que puedas empezar a escribir código sin mucho esfuerzo previo.

En esta Guía rápida se detalla cómo usar las opciones disponibles y la disposición de la estructura de carpetas.

Caracteristicas

  • Live Server
  • Compilar SASS
  • Fuente: Roboto
  • Íconos de Material Design
  • 100% personalizable

Estructura de carpetas

  • /assets Recursos gráficos (imágenes, fuentes, etc)
  • /css Carpeta de archivos CSS compilados
  • /pages Páginas adicionales
  • /scss
    • /components Componentes (No olvides importarlos en main.scss)
    • _init.sscs Inicializa algunas propiedades y etiquetas
    • _variables.sscs Declara e inicializa tus variables
    • main.sscs Archivo SASS que importa los otros y se compila
  • .gitignore Especifica qué archivos o carpetas ignora git
  • index.html Recursos gráficos
  • package.json Archivo de configuración, desde acá puedes registrar más paquetes
  • README.md Guía de instalación

Uso

Live Server

Live Server crea un servidor local y lo abre en tu navegador. También actualiza la página cada vez que encuentra un cambio en el código. Para activarlo, ejecuta el siguiente comando en una nueva terminal.

npm run server

Compilar SASS

El archivo /scss/main.scss compila a /css/main.css. Para compilar automáticamente y escuchar los cambios en el código, ejecuta el siguiente comando en una nueva terminal. 

npm run scss

Fuente: Roboto

Roboto es la fuente instalada por defecto con las siguientes variaciones: Normal y Bold.

Íconos de Material Design

Para mostrar íconos utiliza el siguiente formato, y accede a Material Design Icons para obtener los nombres de los íconos.

home menu folder photo_camera 
<i class="material-icons"> nombre_del_icono </i>

100% personalizable

Todas las características expuestas son modificables. Puedes agregar, editar o eliminar archivos sin problemas, siempre respentado la estructura de carpetas y el lenguaje.

Extra

Usar imágenes para prueba

Si necesitas colocar una imagen y no tienes una a la mano o no quieres hacer todo el proceso para importarla, entonces puedes utilizar Lorem Picsum. Lorem Picsum es una herramienta muy útil que brinda imágenes totalmente a medida por filtros, tamaños, categoría, etc. Visita su página oficial para más información.

Ejemplo con una imagen aleatoria de 300px de ancho y 200px de alto

<img src="https://picsum.photos/300/200" />
